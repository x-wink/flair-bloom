# 气质花 FlairBloom · 剑三自动按键小助手

> 帮你"自动按键"：等CD、平A、采集、连招……让手指歇会儿。免费、离线、不改游戏。

你按下某个键，气质花就帮你**飞快地连按**目标键。剑三里那些要反复戳键盘的活，交给它。

---

## 能帮剑三玩家干嘛

- 🅵 **武学助手 FFF**：开了武学助手只管狂按 F？让它替你自动连 F，手指解放
- 🎯 **一键宏启动**：一长串技能宏，按一下就持续触发，不用反复戳
- 🔁 **手搓循环**：输出循环要一直按同一个键时，按住就交给它连
- 🧩 **多段宏分组**：几段宏之间一键切换、互不打架（见下方"连招不打架"）
- 🌿 顺带：采药挖矿、钓鱼、定时嗑药这类重复点击也能挂

> ⚠️ 它靠"模拟你按键"工作，存在被游戏反作弊检测的风险。用不用、用在哪，请自己权衡（见末尾）。

## 三步上手

1. **装**：去[下载页](../../releases)下最新 `.exe`，双击安装、打开，首次同意协议。
2. **加一条规则**：在「按压连发」页点「+」，点输入框后**直接按你要连的那个键**（键盘 / 鼠标 / 滚轮都行），间隔填 `50`（新手够用），勾上"启用"。
3. **开总开关**：点右下角「全局已禁用」让它变「已启用」，背景变色就生效了——按住那个键自动连发，松手停。

🎉 会了。想玩花样往下看。

## 界面长这样

**主面板**（竖版规则列表）：

```
┌────────────────────────────────────────────┐
│ 🌸 气质花按键助手        ▭   ☰   —   ✕      │ ← 标题栏：▭ 切布局 · ☰ 菜单 · — 收进悬浮窗 · ✕ 关闭
├────────────────────────────────────────────┤
│   按压连发 0/1    │   切换连发 1/1          │ ← 两种连发（数字 = 已启用/总数）
├────────────────────────────────────────────┤
│ ⠿ ┌──────────────────────────────────┐ ✕  │
│   │ 连发按键 [ Q ]   间隔 [ 50 ]ms  ☑ │    │ ← 规则卡（⠿ 拖动排序，☑ 启用）
│   │  ▾ 高级设置                       │    │
│   └──────────────────────────────────┘    │
│ + 添加按压连发规则                          │
├────────────────────────────────────────────┤
│  默认配置 ▾     游戏模式 ▾    [全局已禁用]  │ ← 配置切换 · 输入模式 · 总开关
└────────────────────────────────────────────┘
```

- **标题栏四按钮**：▭ 切换竖版 / 横版布局 · ☰ 菜单 · — 收进右下角悬浮窗 · ✕ 关闭（按住空白处可拖窗）
- **☰ 菜单**：设置、主题颜色、检查更新、诊断修复、关于
- **底部三按钮**：左=配置切换，中=输入模式，右=连发总开关

**横版键鼠图**（点 ▭ 切换）：在键盘 / 鼠标示意图上直接点键设连发，一眼看清哪些键在用——

```
┌──────────────────────────────────────────────┐
│  Esc  F1 F2 F3 …                              │
│  Q  W  E [R] T  Y …   ← 点某个键：切换连发→按压→取消
│  A [S] D  F  G  H …      高亮 = 正在连发
│  鼠标  左 右 中 侧1 侧2  滚轮↑↓                │
└──────────────────────────────────────────────┘
```

**悬浮窗**（点 — 收起后）：桌面右下角一个常驻置顶小条，可拖动——

```
        ┌──────────────────┐
        │ 🌸  Q  F    ▶  ⤢ │ ← Q/F=正在连发的键（呼吸高亮）· ▶ 启动/暂停 · ⤢ 展开主面板
        └──────────────────┘
```

## 玩游戏必看：用"游戏模式"

**游戏里请用「游戏模式」**；「通用模式」只适用于普通软件、网页等非游戏场景，游戏里通常不生效。开启游戏模式：

1. 点底部「输入模式」→ 选「**游戏模式**」。
2. 点"安装"，弹出的授权点同意（黑框一闪是正常的）。
3. **重启电脑**，重新打开气质花、再选一次游戏模式即可。

> 游戏模式会装一个小驱动、并以管理员运行，这样才能把按键送进游戏。装一次以后一直有效。

## 多玩点花样

**两种连发**
- **按压连发**：按住就连、松手就停（适合平A、等CD戳键）
- **切换连发**：按一下开始、再按一下停（适合长时间挂机、采集）；"开始键"和"被连的键"还能设成两个不同键

**所有键都能连**：键盘、鼠标左 / 右 / 中 / 两个侧键、滚轮上下，随便组合（比如"按住侧键连左键""按 F 连滚轮下滚"）。

**两种看法**：点标题栏的布局按钮，可在「竖版规则列表」和「横版键鼠图」间切换——横版直接在键盘 / 鼠标示意图上点键设连发（点一下切换连发、再点改按压、再点取消），一眼看清哪些键在用。（DD驱动模式下请用竖版）

**多套配置**：不同角色 / 心法各存一套，底部左下角点配置名一键切换，互不串。

**连招不打架**：把几条「切换连发」规则放进同一个分组，组内同一时刻只跑一条——比如做多段宏时，按 2 会自动停掉按 1 那条。操作：在「切换连发」页点「+ 新建互斥分组」，把规则拖进去。

**全局热键**：在 设置 → 热键 里设"一键开关所有连发""收起 / 唤出窗口"，游戏里也能直接按（只能用键盘键）。

**主题随心**：点 ☰ 菜单 →「主题颜色」，一键切 明亮 / 黑暗 / 跟随系统，还能挑 21 个门派配色（万花霞紫、天策赤焰、纯阳天青、长歌空青……）。

**声音播报**：可开语音提示，开 / 关、启 / 停时念出来，不用盯着屏幕；四个播报时机可分别开关。

**后台常驻**：关窗口可选「切换到悬浮窗」——收起为桌面右下角一个小悬浮条，常驻置顶、显示正在连发的键、带「启动 / 暂停」开关，点一下就能展开回主面板；连发照常跑，支持开机自启。正在连发的规则卡片也会"呼吸"闪动，一眼看出哪条在跑。

## 常见问题

**剑三里没反应？** 先确认右下角是"全局已启用"、规则已勾选；剑三必须用"游戏模式"（装驱动 + 重启）。

**连太快 / 松手了还在按？** 间隔别填太小，建议 50ms 左右；多条一起连，软件会自动控速，不会停不下来。

**会被封号吗？** 模拟按键有被检测的风险，能不能用、会不会处罚，**请自行评估承担**。个别反作弊还会拦驱动导致进不去游戏，遇到就切回"通用模式"。

**打开时被 Windows 拦（SmartScreen）？** 点"更多信息 → 仍要运行"（暂未做签名）。

**旧配置还能用吗？** 能，覆盖安装直接兼容。

## 稳不稳 · 安不安全

> 这里说的是**软件本身**；会不会被游戏判定违规是另一回事（见开头提醒）。

**稳定**

- **松手就停，不"收不住"**：多条规则一起连也会自动控总速，不会越叠越快、停不下来
- **不卡键**：连发在独立线程跑，万一出意外会自动把按下的键"抬起来"，避免卡死
- **急停彻底**：停止 / 切配置 / 退出都会清干净残留按键

**放心**

- **不改游戏、不读内存**：只在系统层面监听和模拟按键，不碰游戏文件
- **本地离线**：配置加密存在本机，不上传云端，除检查更新外不联网
- **更新有校验**：自动更新走 HTTPS 并校验数字签名，防掉包
- **不记隐私**：本地日志不含账号、硬件等个人信息
- **开源可查**：协议见文末
