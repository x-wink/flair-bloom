# 气质花 FlairBloom

> 面向游戏的按键连发助手 · 轻量 · 无边框 · 本地离线 · 配置加密

气质花是一款运行在 Windows 上的按键连发工具。它**不修改任何游戏文件**，而是通过全局键盘 / 鼠标监听，在你按下触发键时自动、稳定地连续输出目标按键，帮你在游戏中解放手指——等技能 CD、平 A 输出、挖矿、刷材料都能交给它。

核心功能完全免费，配置以加密格式保存在本地，无需联网（仅自动更新会访问网络）。

## 🚀 5 分钟新手教程

跟着下面四步，几分钟就能配好你的第一条连发规则。

### 第 1 步：安装并启动

1. 前往 [Releases](../../releases) 页面，下载最新版本的安装包（`.exe`），双击按提示安装。
2. 从桌面快捷方式或开始菜单打开「气质花按键助手」，窗口默认居中显示。
3. 首次启动会弹出**用户协议**，请滚动到底部、勾选同意后进入主界面（这是免责须知，务必阅读）。

### 第 2 步：认识界面

主界面分为三块——上方标题栏、中间规则区、底部控制栏：

```
┌──────────────────────────────────────────────────┐
│ 🌸 气质花按键助手  [显隐: F8]        ☰   —   ✕   │  ← 标题栏（可拖动移动窗口）
├──────────────────────────────────────────────────┤
│   按压连发  0/1   │   切换连发  1/1               │  ← Tab 栏（数字 = 已启用/总数）
├──────────────────────────────────────────────────┤
│  ⠿ ┌──────────────────────────────────────┐  ✕  │
│    │ 连发按键 [ Q ]      间隔 [ 50 ] ms   ☑ │     │  ← 规则卡片
│    │  ▾ 高级设置                            │     │
│    └──────────────────────────────────────┘     │
│  + 添加按压连发规则                               │
├──────────────────────────────────────────────────┤
│  默认配置 ▾      游戏模式 ▾      [ 全局已禁用 ]   │  ← 底部控制栏
└──────────────────────────────────────────────────┘
```

- **☰ 菜单**：设置、检查更新、更新公告、用户协议、诊断修复、关于
- **— / ✕**：最小化 / 关闭窗口
- **底部三按钮**：配置文件切换、输入模式切换、全局总开关

### 第 3 步：添加你的第一条规则

以最常用的「按住 Q 自动连按 Q」（按压连发）为例：

1. 切到「**按压连发**」Tab。
2. 点击「**+ 添加按压连发规则**」。
3. 点击「**连发按键**」输入框，框内提示「按下按键…」时，**直接按一下你要连发的键**（如键盘 Q，或鼠标左键、侧键，甚至滚轮上/下都行）。
4. 在「**间隔**」里填毫秒数，数值越小连发越快。**新手建议先填 `50`**（约每秒 20 次），稳定又不容易被游戏限流。
5. 勾选卡片右侧的「**启用**」复选框。

> 💡 录入鼠标按键：在「按下按键…」状态下直接按鼠标键即可，右键不会弹出系统菜单。想取消录入，点卡片其它地方就行。

### 第 4 步：打开总开关，开始连发

点击右下角的「**全局已禁用**」按钮，它会变成「**全局已启用**」，整个界面背景同时由蓝灰转为橙色——这就是「连发已生效」的信号。

现在按住 Q（或你设的键），它就会自动连发；松手即停。再次点击总开关即可全部暂停。

🎉 完成！你已经会用气质花了。想玩更多花样，继续看下面的核心功能。

## ✨ 核心功能

### 两种连发模式

| | 按压连发（Hold） | 切换连发（Toggle） |
|---|---|---|
| **触发方式** | 按住触发键期间持续连发，松手即停 | 按一下热键开始连发，再按一下（或按停止键）停止 |
| **典型场景** | 等技能 CD 好就连按（如反复按 Q） | 长时间持续输出（平 A、挖矿、自动钓鱼） |
| **默认行为** | 触发键 = 连发键，按住即连按自身 | 启动热键与连发键可不同，按一下解放手指 |
| **高级选项** | 可单独设「按压键」与「连发按键」（异键触发） | 可单独设「停止热键」 |

> 点击规则卡片底部的「**高级设置**」即可展开异键触发 / 独立停止热键的配置。

### 全键位支持：键盘 + 鼠标 + 滚轮

触发键和连发键都能绑定：

- **键盘**：所有刻字键、F1–F24、小键盘、OEM 标点、方向键与编辑键、CapsLock / NumLock 等
- **鼠标**：左键 / 右键 / 中键 / 侧键 1 / 侧键 2
- **滚轮**：上滚 / 下滚

键盘键、鼠标键、滚轮可自由组合（如「按住侧键连发左键」「按 F 连发滚轮下滚」）。

### 互斥分组（切换连发专属）

把几条切换连发规则拖进同一个「**互斥分组**」后，组内规则**同一时刻只有一条在跑**——开启其中一条会自动停掉同组其它正在连发的规则。适合「武器 / 形态 / 连招」之间一键切换，互不打架。

操作：在「切换连发」Tab 点「**+ 新建互斥分组**」，再把规则卡片拖进分组框；分组可折叠、可重命名（铅笔图标）、可解散（解散后规则保留，只是不再互斥）。

### 规则拖拽排序

每张规则卡片左侧有拖拽手柄「⠿」，按住即可上下拖动调整顺序，也能在分组之间拖入拖出。

### 多配置文件

不同游戏 / 角色可各存一套规则，互不干扰：

- 点底部左下角的**配置名称**（如「默认配置」）展开菜单，点任一配置即可切换。
- 「**新建配置…**」基于当前配置复制一份并自定义名称；「**管理配置…**」进入设置面板，可重命名、删除、导入。
- **默认配置保护**：内置的「默认配置」是出厂预设，不可改名/删除；当你直接修改它时，软件会自动复制出一份新配置（如「我的配置」）并切过去，避免污染基线。

### 全局热键

在「设置 → 热键」可绑定三个全局热键，**不占用界面焦点、随时生效**：

- **全局开启 / 停止键**：一键开关所有连发
- **面板显隐键**：把窗口收进任务栏 / 唤回前台

> 全局热键只能绑定**键盘**按键，且三者不能相同——这是为了避免按键被某个热键抢走、导致其它功能失灵。

### 声音播报

在「设置 → 声音」可开启语音提示：开启 / 关闭总开关时播报，切换连发启停时播报对应按键名。语句、语速、音调、音量、系统语音都可自定义并试听。

### 激活态视觉反馈

规则**正在连发**时，卡片会显示**呼吸脉冲动画**，让你一眼区分「已启用（待命）」和「正在连发（运行中）」。

### 系统托盘常驻

关闭窗口时可选择「最小化到托盘」，程序在后台继续运行、连发保持有效。托盘菜单提供全局开关、打开面板、退出；双击托盘图标或再次启动程序都能唤回窗口。也支持开机自启。

## 🛡️ 为什么稳定、放心用

> ⚠️ 先说清楚：本工具靠**模拟输入**工作，部分游戏的反作弊系统可能检测此类行为。它**无法保证不被检测或封号**，是否使用请自行评估风险（详见[注意事项](#-常见问题与注意事项)）。下面讲的是**软件本身**在稳定性、隐私与数据安全上的设计。

- **不改游戏文件、不注入进程**：只在系统层面监听与模拟按键，不读写、不挂钩游戏内存，不碰游戏安装目录。
- **本地离线**：所有功能本地完成，配置不上传云端；除「检查更新」外不主动联网。
- **配置加密存储**：配置文件以 `.qzh` 格式用 **AES-256-GCM** 加密，文件头带防篡改校验，损坏或被改动会被识别。
- **松手就停，绝不"收不住"**：多条规则同时连发时，软件会自动平摊总速率（总速度与规则条数无关），从根上避免了"叠加太快、松手后还在狂按停不下来"的问题。
- **崩溃也不卡键**：连发在独立线程运行并被异常捕获包裹，万一线程意外退出，会按内部账本自动补发"抬键"，最大限度避免按键卡死。
- **紧急停止 100% 生效**：停止 / 暂停 / 切换配置 / 退出时会彻底复位内部按键状态，清理可能残留的"幻影按键"。
- **更新有签名校验**：自动更新走 HTTPS，并强制校验更新包的数字签名，杜绝被中间人替换。
- **日志不含隐私**：本地日志不记录硬件 ID、用户名等个人信息，排查问题时可放心提供。
- **开源可查**：采用 [CC BY-NC-SA 4.0](./LICENSE) 协议开源，行为透明可审阅。

## 🎮 输入模式怎么选

不同游戏对模拟输入的过滤策略不同，气质花提供三档输入通道。点底部「**输入模式**」按钮切换。

| 模式 | 底层 | 适用场景 | 要求 |
| --- | --- | --- | --- |
| **游戏模式**（推荐 ⭐） | Interception 内核驱动 | 使用 Raw Input 的多数游戏（含**剑网三**），**首选** | 安装驱动 + **重启电脑**，以管理员运行 |
| **通用模式** | 系统 `SendInput` | 最简单、免驱动，但很多游戏不响应 | 无 |
| **DD驱动**（备用） | ddxoft DD 虚拟键鼠 | 仅当游戏模式仍无效时再尝试 | 以管理员运行 |

> **建议优先用「游戏模式」**，它兼容性与稳定性最好。「DD驱动」仅作备用——DD 系列驱动在部分场景下可能停不住或自停连发，稳定性不及前者。原「DDHID」模式经测试不稳定（可能蓝屏），**已禁用**，待新版内测稳定后再考虑开放。

### 启用游戏模式（Interception）

1. 底部「输入模式」→ 选「**游戏模式**」。
2. 弹出「安装驱动」提示，点「安装」并通过 UAC 授权（控制台窗口一闪而过即安装完成）。
3. **重启电脑**，重新打开气质花再切到游戏模式即可生效。

### 启用 DD驱动

1. 底部「输入模式」→ 选「**DD驱动**」。
2. 若当前不是管理员身份，软件会请求 UAC 提权并重启自身。
3. 重启后自动切到 DD驱动模式，**无需安装额外驱动**（DD DLL 随应用打包）。

### 卸载驱动

打开「**☰ 菜单 → 诊断修复**」可卸载游戏模式驱动或残留的 DDHID 驱动。卸载 Interception 后**需重启电脑**才彻底生效。

## 💻 系统要求与下载安装

- **系统**：Windows 10 / 11（64 位）
- **运行时**：无需额外安装，双击即用
- **权限**：使用「游戏模式」或「DD驱动」需以管理员身份运行

下载：前往 [Releases](../../releases) 下载最新 `.exe` 安装包，运行后按提示安装。首次启动需同意用户协议。

> 若 Windows SmartScreen 拦截，点「更多信息 → 仍要运行」即可（早期版本尚未做代码签名）。

## ⚙️ 设置面板

点「**☰ 菜单 → 设置**」打开，分四个分区：

- **通用**：开机自启、关闭按钮行为（最小化到托盘 / 直接退出）
- **热键**：全局开启 / 停止键、面板显隐键
- **声音**：语音播报开关、语句、语速、音调、音量、系统语音、试听
- **配置文件**：配置卡片一览，支持切换、重命名、删除、导入外部配置

> **导入外部配置**：支持从「丐帮高手」（`config.json`）一键导入，自动扫描桌面 / 下载目录，识别连发规则与热键后预览确认，导入为独立配置文件。

「**诊断修复**」（☰ 菜单内）则展示管理员权限、开机自启、当前输入模式、驱动状态等运行状况，并提供驱动安装 / 卸载与安装前置检查。

## ❓ 常见问题与注意事项

**Q：按键在游戏里没反应？**
先确认右下角是「全局已启用」且规则已勾选启用。仍无效多半是该游戏过滤了 `SendInput`，请切到「**游戏模式**」（按上文安装驱动并重启，剑网三等 Raw Input 游戏即可生效）；若游戏模式仍无效，再尝试「**DD驱动**」。通用模式下若连界面外都监听不到按键，试试**以管理员身份运行**。

**Q：连发太快被游戏限流 / 松手后短暂还在按？**
连发间隔默认与最低均为 **10ms**（约每秒 100 次）。建议日常用 **50ms** 左右（约 20 次/秒）。多条规则同时连发时软件会自动平摊总速率，避免叠加超发。

**Q：会被游戏封号吗？**
本工具通过模拟输入工作，**部分游戏反作弊会检测此类行为**，可能导致账号处罚。是否使用、用在何处，请**自行评估并承担风险**。部分反作弊（如 EA Anti-Cheat）还会检测 Interception / DD 驱动并阻止游戏启动，遇到时请切回「通用模式」。

**Q：装了游戏模式驱动后登录界面键盘没反应？**
极少数 Windows 11 设备上 Interception 驱动可能导致此问题，可进安全模式卸载驱动恢复。

**Q：旧配置还能用吗？**
能。覆盖安装新版即可，配置完全兼容；旧配置里低于 10ms 的间隔会在加载时自动调整为 10ms。
