# 气质花 FlairBloom

> 面向游戏的按键连发助手 · 轻量 · 无边框 · 离线可用

---

## 功能简介

气质花是一款运行在 Windows 上的按键连发工具，无需修改游戏文件，通过全局键盘监听模拟按键输出，帮助你在游戏中解放手指。

| 功能 | 说明 |
| --- | --- |
| **按压连发** | 按住触发键时，自动持续输出目标键 |
| **切换连发** | 按下热键后连发持续运行，再次按下（或停止热键）停止 |
| 多规则管理 | 同时配置多条规则，按 Tab 分组管理 |
| 全局开关 | 一键暂停 / 恢复所有连发，状态通过界面颜色直观反映 |
| 无边框界面 | 透明圆角窗口，可在桌面任意拖动 |

## 使用说明

### 启动程序

安装完成后双击桌面快捷方式或从开始菜单打开「气质花」，程序窗口默认居中显示。

### 界面说明

```
┌─────────────────────────────────┐
│ 🌸 气质花按键助手 v0.1    — ✕  │  ← 标题栏（可拖动）
├─────────────────────────────────┤
│  按压连发  0/1 │ 切换连发  1/1  │  ← Tab 栏（数字 = 启用数/总数）
├─────────────────────────────────┤
│  ┌──────────────────────────┐   │
│  │ 按压键  →  连发按键  间隔 │●│  │  ← 规则卡片（右侧绿点 = 已启用）
│  └──────────────────────────┘   │
│  + 添加按压连发规则              │
├─────────────────────────────────┤
│  恢复默认          全局开关 [已启用] │
└─────────────────────────────────┘
```

### 添加规则

1. 点击底部「**+ 添加规则**」按钮
2. 点击「**按压键**」输入框，然后按下你想监听的键（如 `Q`）
3. 点击「**连发按键**」输入框，按下要连发输出的键（默认与按压键相同）
4. 调整「**间隔**」（毫秒），数值越小连发越快，最低 10ms
5. 右侧拨动开关启用该规则

### 按压连发 vs 切换连发

| | 按压连发（Hold） | 切换连发（Toggle） |
|---|---|---|
| **触发方式** | 按住触发键期间持续连发 | 按下热键后一直连发，再按停止 |
| **适用场景** | 需要精确控制时机（技能、跳跃） | 长时间持续输出（平 A、挖矿） |
| **高级选项** | — | 可设置独立「停止热键」 |

> **切换连发高级设置**：点击规则卡片底部「高级设置」可展开，配置与启动热键不同的停止热键。

### 全局开关

点击右下角「**已启用 / 已禁用**」按钮，可一键暂停或恢复**所有规则**。  
界面背景颜色会同步变化：
- 🔵 蓝灰色 = 全局已禁用
- 🟠 橙色 = 全局已启用

### 关闭窗口

首次点击右上角 ✕ 时，会弹出选择框：

- **最小化到托盘**（推荐）：程序继续在后台运行，连发功能保持有效
- **直接退出**：关闭程序，所有连发停止

勾选「记住我的选择」后，下次直接执行，不再弹窗。

### 恢复默认配置

点击左下角「**恢复默认**」按钮，可清空当前所有规则并重置为内置默认配置。

---

## 注意事项

- 本程序通过模拟键盘输出工作，部分游戏的反作弊系统可能检测此类行为，**请自行评估风险**
- 连发间隔最低 10ms，建议不低于 **50ms**（约 20次/秒），过低可能被系统或游戏限流
- 若程序无法监听按键，请尝试**以管理员身份运行**
