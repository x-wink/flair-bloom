# 气质花 FlairBloom

> 面向游戏的按键连发助手 · 轻量 · 无边框 · 离线可用

---

## 功能简介

气质花是一款运行在 Windows 上的按键连发工具，无需修改游戏文件，通过全局键盘监听模拟按键输出，帮助你在游戏中解放手指。

| 功能 | 说明 |
| --- | --- |
| **按压连发** | 按住触发键时，自动持续输出目标键 |
| **切换连发** | 按下热键后连发持续运行，再次按下（或停止热键）停止 |
| 多规则管理 | 同一配置内可同时配置多条规则，按 Tab 分组管理 |
| 多配置文件 | 不同游戏 / 角色独立保存一套规则，下拉菜单一键切换 |
| 三档输入模式 | 通用 / 游戏 / 究极HID 自由切换，覆盖不同游戏的输入过滤 |
| 全局开关 | 一键暂停 / 恢复所有连发，状态通过界面颜色直观反映 |
| 激活态反馈 | 规则正在连发时卡片呼吸脉冲，一眼区分「已启用」和「正在连发」 |
| 自动更新 | 后台静默下载新版本，下次启动自动安装 |
| 无边框界面 | 透明圆角窗口，可在桌面任意拖动 |

---

## 系统要求

- Windows 10 / 11（64 位）
- 无需安装运行时，双击即用
- 启用「游戏模式」或「究极HID」需以管理员身份运行

## 使用说明

### 启动程序

安装完成后双击桌面快捷方式或从开始菜单打开「气质花」，程序窗口默认居中显示。

### 界面说明

```
┌─────────────────────────────────────────────┐
│ 🌸 气质花按键助手 v0.1.8       ☰  —  ✕     │  ← 标题栏（可拖动）
├─────────────────────────────────────────────┤
│  按压连发  0/1   │   切换连发  1/1          │  ← Tab 栏（数字 = 启用数/总数）
├─────────────────────────────────────────────┤
│  ┌────────────────────────────────────┐     │
│  │ 连发按键   间隔        ▢ 启用      │     │  ← 规则卡片
│  │  ▾ 高级设置                        │     │
│  └────────────────────────────────────┘     │
│  + 添加按压连发规则                          │
├─────────────────────────────────────────────┤
│  默认配置 ▾   输入模式 通用模式 ▾   全局开关 [已启用] │
└─────────────────────────────────────────────┘
```

- **☰** 菜单：检查更新、更新公告、用户协议、关于、卸载驱动（关于弹窗内整合诊断信息与「打开安装目录 / 数据目录 / 日志目录」入口）
- **— ✕** 最小化 / 关闭

### 添加规则

1. 切换到「**按压连发**」或「**切换连发**」Tab
2. 点击底部「**+ 添加按压连发规则 / + 添加切换连发规则**」按钮
3. 点击「**连发按键**」输入框，按下要连发输出的键
4. 调整「**间隔**」（毫秒），数值越小连发越快，最低 10ms
5. 右侧勾选启用该规则

### 按压连发 vs 切换连发

| | 按压连发（Hold） | 切换连发（Toggle） |
|---|---|---|
| **触发方式** | 按住触发键期间持续连发 | 按下热键后一直连发，再按停止 |
| **适用场景** | 等技能 CD 好就连按（如 Q 技能） | 长时间持续输出（平 A、挖矿） |
| **默认行为** | 触发键 = 连发按键，按住即连按自身 | 启动热键独立，连发自定义按键 |
| **高级选项** | 可独立设置「按压键」与「连发按键」 | 可独立设置「停止热键」 |

> 点击规则卡片底部「**高级设置**」可展开异键触发或独立停止热键的配置。

### 激活态视觉反馈

当某条规则正在连发时（按压模式触发键被按住、切换模式已开启），规则卡片会显示**呼吸脉冲动画**，便于一眼区分「已启用」与「正在连发」。

### 全局开关

点击右下角「**已启用 / 已禁用**」按钮，可一键暂停或恢复**所有规则**。  
界面背景颜色会同步变化：
- 🔵 蓝灰色 = 全局已禁用
- 🟠 橙色 = 全局已启用

### 多配置文件管理

不同游戏或角色可使用独立的配置文件，互不干扰。

- 点击底部左下角的**配置名称**（如「默认配置」）展开下拉菜单
- 菜单内可：
  - **切换配置**：点击列表中任一配置切换
  - **新建配置…**：基于当前激活配置复制一份，自定义名称
  - **重命名当前配置…**：改名（默认配置除外）
  - **删除当前配置**：删除后自动回退到默认配置（默认配置除外）
- **默认配置保护**：内置 `defults` 配置不可重命名或删除；当你直接修改默认配置时，应用会自动 fork 一份新配置（如「我的配置」）并切换激活，避免污染基线

### 关闭窗口

首次点击右上角 ✕ 时，会弹出选择框：

- **最小化到托盘**（推荐）：程序继续在后台运行，连发功能保持有效
- **直接退出**：关闭程序，所有连发停止

勾选「记住我的选择」后，下次直接执行，不再弹窗。

## 输入模式

气质花提供三档输入注入通道，应对不同游戏的反作弊与输入过滤策略。点击底部「**输入模式**」按钮即可切换。

| 模式 | 底层 | 适用场景 | 特殊要求 |
| --- | --- | --- | --- |
| **通用模式** | `SendInput` | 默认，最稳定，绝大多数应用通用 | 无 |
| **游戏模式** | Interception 内核驱动 | 使用 Raw Input API 的多数游戏 | 安装驱动 + **重启电脑** |
| **究极HID** | ddxoft DD 虚拟 HID 驱动 | Interception 仍无效的游戏（如**剑网三**） | 安装驱动 + **管理员运行** |

### 启用游戏模式（Interception）

1. 点击底部「输入模式」→ 选择「游戏模式」
2. 弹出「安装驱动」提示，点击「安装」并通过 UAC 授权
3. 控制台一闪而过表示安装完成
4. **重启电脑**，再次打开气质花切换到游戏模式即可生效

### 启用究极HID（DD-HID）

1. 点击底部「输入模式」→ 选择「究极HID」
2. 弹出「安装驱动」提示，点击「安装」并通过 UAC 授权
3. 控制台一闪而过表示安装完成（**无需重启**）
4. 若当前不是管理员身份，应用会请求 UAC 提权重启自身，重启后自动切换到该模式

> 究极HID 模式下，「切换连发」的目标键不能与启动 / 停止热键相同（DD 驱动无法标识自身注入的按键）；「按压连发」不受此限制。

### 卸载驱动

打开右上角 **☰ 菜单 → 卸载驱动**，可分别卸载游戏模式驱动或究极HID 驱动。

- 卸载 Interception 后**需重启电脑**才能彻底生效
- 卸载 DD-HID 后**无需重启**

也可以管理员身份运行命令行：

```
install-interception.exe /uninstall    # Interception
ddc.exe -u                              # DD-HID
```

## 自动更新

启动应用时会在后台静默检查并下载新版本。下载完成后：

- 标题栏右上角 **☰ 菜单 → 更新公告** 旁显示红点提示
- 点击「更新公告」可查看本次更新的版本说明
- **下次启动时自动完成安装**，无需手动操作

如果想立即检查更新，可点击 **☰ 菜单 → 检查更新**。

## 注意事项

- 本程序通过模拟键盘输出工作，部分游戏的反作弊系统可能检测此类行为，**请自行评估风险**
- 连发间隔最低 10ms，建议不低于 **50ms**（约 20 次 / 秒），过低可能被系统或游戏限流
- 通用模式下若程序无法监听按键，请尝试**以管理员身份运行**
- Interception 驱动在极少数 Windows 11 设备上可能导致登录界面键盘无响应，可通过安全模式卸载恢复
- 部分游戏反作弊（如 EA Anti-Cheat）会检测 Interception 或 DD-HID 驱动并阻止游戏启动；遇到这种情况请切回「通用模式」
