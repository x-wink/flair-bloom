# 更新日志

所有版本的更新记录。发版前在 `[Unreleased]` 节填写内容，运行 `pnpm bump-version` 后自动转为对应版本节。

## [Unreleased]

## [0.1.3] - 2026-05-25

### 新功能

- **驱动增强模式**：新增 Interception 内核驱动支持，解决剑网三等使用 Raw Input API 的游戏中按键模拟无效的问题。在面板底部点击「驱动增强」开关即可启用，首次使用需安装驱动并重启电脑
- **双模式架构**：标准模式（SendInput）与驱动模式可自由切换，默认标准模式，兼容性最佳；驱动模式兼容更多游戏

### 行为变更

- 用户协议更新至 v1.1，新增第三方驱动组件声明与风险提示，升级后需重新确认协议

### 升级方式

覆盖安装新版安装包即可。如需使用驱动增强功能，点击面板底部「驱动增强」开关，按提示安装驱动并重启电脑。

### 已知问题

- Interception 驱动在极少数 Windows 11 设备上可能导致登录界面键盘无响应，可通过安全模式卸载恢复
- 部分游戏反作弊（如 EA Anti-Cheat）会检测 Interception 驱动并阻止游戏启动

## [0.1.2] - 2026-05-25

本次重点修复了自动更新流程，并完善了更新体验。

### 新功能

- **自动更新**：新版本发布后，应用将在后台静默下载安装包，下次启动时自动完成安装，无需手动操作
- **更新公告**：新增「更新公告」弹窗，展示版本说明；下拉菜单可随时回看，有待安装更新时以红点提示

### 问题修复

- 修复版本号显示：关于页面版本号现从系统读取，不再与实际版本不符
- 修复扩展键（方向键、Insert、Delete 等）注入时编码错误导致按键无效的问题
- 修复按住时长精度问题，`hold_ms` / `rest_ms` 控制更准确
- 修复引擎多次重建可能挂载双重键盘钩子的问题
- 修复退出时极端情况下可能遗漏 key_up 的竞态问题

### 升级方式

覆盖安装新版安装包即可，配置文件完全兼容，无需重新设置规则。

## [0.1.1] - 2026-05-25

本次为紧急修复版本，解决了 v0.1.0 中连发功能在游戏内无效的根本性问题。

### 问题修复

- **连发在游戏中完全无效**（v0.1.0 已知问题）：改为硬件扫描码注入，游戏内连发恢复正常，强烈建议所有 v0.1.0 用户立即升级
- **扩展键连发无效**：方向键、Delete、Home、End 等扩展键因扫描码前缀处理错误无法正常连发，本次一并修正
- **停止连发时目标键偶发卡住**：重写了线程退出与按键释放的协调逻辑，确保键位正确抬起
- **切换规则时旧线程短暂残留**：保证新旧线程不再重叠运行

### 升级方式

覆盖安装新版安装包即可，配置文件完全兼容，无需重新设置规则。

## [0.1.0] - 2026-05-24

首个公开内测版（MVP），功能基本可用。

### 新功能

- **连发引擎**：按压连发（Hold）、切换连发（Toggle）、多规则并行、全局开关
- **按键录入**：点击输入框直接按下目标键完成绑定
- **无边框透明窗口**：可在桌面任意拖动
- **系统托盘常驻**：关闭窗口后程序继续在后台运行，连发保持有效
- **开机自启**：支持随系统启动，可在托盘菜单中关闭
- **自动更新**：启动时静默检查新版本（v0.1.2 修复实际可用）
- **配置加密存储**：配置文件以 .qzh 格式存储，采用 AES-256-GCM 加密

### 已知问题

- 游戏内连发无效（已在 v0.1.1 修复）
